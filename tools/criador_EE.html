Regras de Criação

Custo Base: 1 Vigor

1. Abordagem (Visual)

  • Azul (D), Vermelho (O), Verde (E). Sem custo.

2. Efeito Principal

  • Bônus/Ônus principal: Sem custo.
  • Dano: Apenas Dano ao Vigor permitido aqui. Até 2 é grátis. +1 Custo para cada 2 de dano acima de 2.
  • Condição: Custo extra de +2.

3. Efeitos Secundários

  • +1 Custo para cada Bônus ou Ônus adicional.

4. Área de Efeito

  • Alvo Único (até 9m): Sem custo.
  • Área Especial: +1 Custo base.
  • Esfera/Quadrado: Base 1.5m tamanho e 9m alcance. +1 custo/1.5m extra.
  • Linha/Cone: Base 4.5m compr. x 1.5m (linha) ou 4.5m (cone). +1 custo/1.5m extra.
  • Pilar: Base 1.5m raio x 3m altura. +1 custo/1.5m extra.

5. Mitigação

  • -1 Custo para cada forma de resistir (teste, pagar vigor, receber ônus, receber dano).
  • Dano Estresse: Permitido APENAS aqui (Máx 1).
AVISO: Todo efeito especial criado aqui depende da aprovação do Narrador / Mestre para ser utilizado em jogo.
